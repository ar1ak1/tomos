# リリース計画

## Milestones

| Milestone | Goal |
|---|---|
| M1 | Laravel初期構築・認証 |
| M2 | Social login |
| M3 | コミュニティ作成・コミュニティ運営者管理 |
| M4 | イベント作成・公開 |
| M4 | 参加枠・申し込み |
| M5 | Stripe連携・有料申し込み |
| M6 | 事前アンケート |
| M7 | チェックイン・QR |
| M8 | メール通知・定期処理 |
| M9 | 監査ログ・CSV |
| M10 | staging検証 |
| M11 | productionリリース |

## Release Criteria

メール確認必須の登録・ログイン、Google/GitHubログイン、コミュニティ作成/管理者追加、Stripe連携、イベント作成/公開/中止、無料/有料申し込み、Stripe Webhook成功、抽選、無料枠キャンセル待ち、事前アンケート/CSV、QRチェックイン、前日リマインド/中止メール、監査ログ、主要Feature Test、staging確認、productionデプロイ可能。

## Post-MVP

検索、タグ、OpenSearch、事後アンケート、分析、ブラックリスト、有料枠キャンセル待ち、返金連携、通知設定、E2E/VRT、自動削除/匿名化、システム管理画面、外部API。
