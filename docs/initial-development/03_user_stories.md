# ユーザーストーリー

## 参加者

- 開催予定イベント一覧から参加したい勉強会を見つける。
- メール確認済みログインユーザーとしてイベント枠に申し込む。
- 行動規範・ポリシー同意を行い、枠にアンケートがあれば回答する。
- 有料枠では決済完了後に参加確定する。
- 参加確定済みイベントのチケット画面でQRコード、会場詳細、オンラインURLを確認する。
- チケットはスマホ表示・印刷利用の両方を想定する。
- 参加予定と参加履歴を確認する。
- キャンセル期限内に自分の申し込みをキャンセルする。自動返金はされない。

## コミュニティ管理者

- コミュニティを作成する。
- CoC URL、問い合わせメール、キャンセルポリシー、特商法等URLを設定する。
- 既存ユーザーを承認不要でコミュニティ管理者に追加する。
- コミュニティ単位でStripe連携する。
- イベントを作成・公開・中止する。
- 複数枠、事前アンケート、抽選、強制当選/落選を管理する。
- 参加者一覧、アンケート回答CSVを確認する。
- 参加者を手動またはQRでチェックインする。
- 未チェックインの参加確定者をno-showにする。

## イベント個別管理者

- 対象イベントに対して、コミュニティ管理者と同等のイベント管理操作を行う。
- コミュニティ設定、Stripe連携、他イベントは管理できない。
